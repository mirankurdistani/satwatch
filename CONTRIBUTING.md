# Contributing to SatWatch

Thank you for your interest in contributing to SatWatch.

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Run tests: `python -m pytest tests/`
5. Commit: `git commit -m "Add: your feature description"`
6. Push and open a Pull Request

## Areas where help is needed

- Improving conjunction accuracy (higher time resolution)
- Adding support for non-Starlink satellites (OneWeb, Kuiper)
- Machine learning anomaly detection
- Unit test coverage
- Performance optimization for full fleet scanning

## Code style

- PEP 8 compliant Python
- Docstrings for all functions
- Comments in English

## Questions?

Open an issue on GitHub.
